# 🌐 Language Translation Tool
### CodeAlpha AI Internship — Task 1

A desktop-based Language Translation Tool built with Python and Tkinter
that uses the Google Translate API to translate text between 45+ languages.

## Features
- 45+ Languages supported
- Auto Detect source language
- Swap languages instantly
- Copy translation to clipboard
- Text-to-Speech (Speak button)
- Modern dark-themed UI

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python main.py

## Tech Stack
- Python 3.8+
- Tkinter (GUI)
- deep-translator (Google Translate)
- pyttsx3 (Text-to-Speech)
